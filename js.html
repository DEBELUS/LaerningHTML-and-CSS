آموزش JavaScript

1️⃣ مقدمه

JavaScript (JS) یک زبان برنامه‌نویسی است که به صفحات وب قابلیت تعامل و پویا بودن می‌دهد. با JS می‌توان رفتار عناصر HTML را کنترل کرد.

2️⃣ افزودن JS به HTML

می‌توان JS را به سه روش به HTML اضافه کرد: 

<!-- داخلی (Inline) -->
<button onclick="alert('سلام!')">کلیک کن</button>

<!-- درون‌خطی (Internal) -->
<script>
  console.log('سلام دنیا');
</script>

<!-- خارجی (External) -->
<script src="script.js"></script>

3️⃣ متغیرها و انواع داده

در JS می‌توان متغیرها را با let، const یا var تعریف کرد. انواع داده شامل عدد، رشته، بولین و آرایه هستند. 

let name = "کاربر";
const age = 16;
var isStudent = true;
let colors = ["red", "blue", "green"];

4️⃣ توابع (Functions)

توابع برای گروه‌بندی دستورات و استفاده مجدد از آن‌ها کاربرد دارند. 

function greet(name) {
  alert("سلام " + name + "!");
}

greet("کاربر");

5️⃣ شرط‌ها و حلقه‌ها

با شرط‌ها و حلقه‌ها می‌توان جریان برنامه را کنترل کرد: 

if (age > 18) {
  console.log("بالغ");
} else {
  console.log("نوجوان");
}

for (let i = 0; i < 5; i++) {
  console.log(i);
}

7️⃣ دستکاری عناصر HTML

با JS می‌توان محتوا و ویژگی‌های عناصر HTML را تغییر داد: 

document.getElementById("myText").innerText = "متن جدید";
document.querySelector("p").style.color = "red";

7️⃣ رویدادها (Events)

رویدادها به JS اجازه می‌دهند تا به تعامل کاربر پاسخ دهد، مانند کلیک و حرکت موس. 

document.getElementById("btn").addEventListener("click", function() {
  alert("دکمه کلیک شد!");
});

8️⃣ جمع‌بندی

JS زبان اصلی برای پویا کردن صفحات وب است. یادگیری آن به شما اجازه می‌دهد تا وبسایت‌هایی تعاملی و کاربردی ایجاد کنید.
