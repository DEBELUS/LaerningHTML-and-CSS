آموزش CSS

1️⃣ مقدمه

CSS (Cascading Style Sheets) زبان استایل‌دهی به صفحات وب است که برای کنترل ظاهر و طراحی صفحه استفاده می‌شود.

2️⃣ انتخابگرهای CSS

انتخابگرها به ما این امکان را می‌دهند که به تگ‌های HTML مشخصی استایل بدهیم: 

p { color: red; }

3️⃣ ویژگی‌های CSS

ویژگی‌ها (Properties) از طریق آن‌ها می‌توانیم ظاهر عناصر HTML را تغییر دهیم. به عنوان مثال: 

h1 { font-size: 24px; }

4️⃣ رنگ‌ها و پس‌زمینه‌ها

برای تغییر رنگ متن، پس‌زمینه و دیگر خصوصیات استفاده می‌کنیم: 

body { background-color: #f0f0f0; color: #333; }

5️⃣ جعبه‌بندی (Box Model)

هر عنصر در CSS به صورت یک جعبه است که شامل padding، margin، border و content می‌شود. 

div {
  margin: 20px;
  padding: 10px;
  border: 2px solid #000;
}

6️⃣ طراحی ریسپانسیو

با استفاده از media queries می‌توانیم طراحی ریسپانسیو ایجاد کنیم که صفحات وب در دستگاه‌های مختلف به خوبی نمایش داده شوند. 

@media (max-width: 600px) {
  body { background-color: lightblue; }
}

7️⃣ جمع‌بندی

CSS ابزاری قدرتمند برای طراحی و استایل‌دهی به صفحات وب است. با استفاده از ویژگی‌های مختلف آن، می‌توانیم صفحات وب جذاب و کاربردی بسازیم.
